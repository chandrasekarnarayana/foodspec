## Still Stuck?

If you can't find an answer:

1. **Search the FAQ** — Use Ctrl+F to search [faq.md](faq.md) for keywords
2. **Check troubleshooting** — Browse [troubleshooting.md](troubleshooting.md) by category
3. **Search GitHub Issues** — Your issue may already be reported
4. **Ask the community** — Post in GitHub Discussions
5. **Report a bug** — Open a new GitHub Issue with a minimal reproducible example
