# API Reference Hub

This hub links to mkdocstrings-generated API pages for the main FoodSpec modules. Use it alongside the narrative chapters to see exact signatures, parameters, and returns.

## How to navigate
- **Core data structures:** See `datasets.md` for `FoodSpectrumSet` and `HyperSpectralCube`.
- **Preprocessing:** Baseline, smoothing, normalization, cropping, FTIR/Raman helpers in `preprocessing.md`.
- **Features:** Peaks, bands, ratios, fingerprint in `features.md`.
- **Chemometrics:** PCA, model factory, validation, mixture, deep in `chemometrics.md`.
- **Workflows:** High-level helpers (oils, heating, QC, protocols) in `workflows.md`.
- **IO & data:** Library creation/loading, CSV import/export, public loaders in `io.md`.
- **Statistics:** Hypothesis tests, correlations, effect sizes in `stats.md`.
